When finish NestJS will update to:

DataBase 
1) id - uuid(v4)
2) position - position on display list
3) title
4) description
5) priority
   - (1)-mało ważne
   - (2)-standard
   - (3)-ważne
   - (4)-bardzo ważne
6) labels - kategoria
   - (0) Dodaj Własną
   - (1) Brak
   - (2) Zakupy
   - (3) Osobiste
   - (4) Praca
7) isComplete
   - (0)-Nie rozpoczęto
   - (1)-W trakcie
   - (2)-Zakończono
8) startedAt - date
9) deadline - date
10) whereToSave 
    - (1) na wszystkich urządzeniach
    - (2) tylko na tym urządzeniu
